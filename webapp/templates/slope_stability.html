Slope Stability Parameters (Midas GTS NX Input)

Enter soil layer properties from borehole data. Computes derived parameters for Midas GTS NX Mohr-Coulomb material model: General, Porous, and Non-Linear tabs (Mohr 1900; Coulomb 1776; Midas IT 2023 GTS NX User Manual).

QA notes on the column inputs (hover the i icons in the table headers below):
• c (cohesion): Provide a value only if a UCS / unconsolidated-undrained triaxial result is available. Otherwise leave 0 and let the SPT correlation drive c (Polish PN-59/B-03020) — do not enter empirical values for cohesionless soils.
• γsat: Derived as (Gs + e)/(1 + e) · γw with e = Gs·w (saturated state). If your moisture content is field-not-saturated, γsat in the report is theoretical, not measured.
• Porous tab (kx, ky, kz): Currently set to a constant 1×10−5 m/s default. Replace with project-specific permeability from falling-head / pumping tests when available.
• Dilatancy ψ = max(φ − 30°, 0): Bolton (1986) empirical relation — valid only for medium-dense to dense sands. For clays / loose sands set ψ = 0.
• Tensile strength: Set to 0 (soil cannot sustain tension; tension cut-off in Mohr-Coulomb).
• Interface reduction Rinter: Not auto-computed — for soil-structure interaction in Midas GTS NX use Rinter = 0.6–0.8 for sand/concrete, 0.5 for clay/concrete (Plaxis Material Models Manual).
